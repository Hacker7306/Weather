# Weather


**This site shows the current weather of a place that has been taken from open-metro api**

It uses fetch latitude and longitude data from geocoding api and then these latitiude and 
longitude are given to open-metro api and then respective datas are fetched from the open-
metro api like :

- Location datas
- Current temperature
- Time
- Weather
- Maximum temperature
- Minimun temperature
- Humidity
- Wind speed
- Uv index
